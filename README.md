# Summary
Repository with scripts for the analysis of LC-MS/MS data generated for the manuscript **Synthesis of guanylylated peptides derived from norovirus VPg as molecular tools for studying norovirus replication** by *Puffelen B van & *Leandro X Neves, _et al._ (2026). 

> *authors contributed equally to this work.

# Data availability
The mass spectrometry proteomics data have been deposited to the ProteomeXchange Consortium (http://proteomecentral.proteomexchange.org) via the PRIDE partner repository [REF doi: 10.1093/nar/gkae1011] with the dataset identifier PXD075074 (PRM) and PXD075200 (synthetic guanylyl peptide). PRM data was also deposited to Panorama Public https://doi.org/10.6069/4zpa-pw40.
